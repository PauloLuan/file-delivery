## NODE VERSION

You should user Node version >= 12 

## Installation

```sh
$ yarn install
```

## Usage

then start the server

```sh
$ yarn start
```

## Unit tests

```sh
$ yarn test
```

Getting the coverage report:
```sh
$ yarn coverage
```
